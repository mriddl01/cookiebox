B) Merge triggers Jenkins, Jenkins tells IPC to build a Jenkins machine
C) server spun up (may/may not happen - not required, but eventually we will want this feature - Plutonium does it, so I know it can be done) 
D) Code is checked out, the appropriate build is done, tests are ran 
E) Yammer notification that build is under way
F) Nagios moniters build
G) Success, image pushed to registry

I) Jenkins spun up 
J) image is pulled out of registry
K) Yammer notified test is under way
L) Nagios monitors containers during test
M) image is tagged in registry
... done
